## 🧠 Problem Understanding & Research

Rocket motion follows **Newton’s Second Law of Motion**:



\[
F = m \times a
\]



A rocket launches because **engine thrust produces upward force**, while **gravity and air resistance oppose the motion**.

### Key Forces Affecting Rocket Launch

- **Thrust** → Force generated by rocket engines pushing the rocket upward.  
- **Gravity** → Earth’s gravitational force pulling the rocket downward.  
- **Drag** → Air resistance slowing the rocket during ascent.  
- **Payload** → Weight of cargo carried by the rocket such as satellites or astronauts.  

Increasing payload increases rocket mass, which requires **greater thrust and fuel for successful launch**.

# 🚀 Rocket Launch Simulation

The project includes a rocket launch physics simulation built using Python.

### Simulation Process

1. Define initial rocket parameters:  
   - rocket mass  
   - payload weight  
   - thrust  
   - drag coefficient  
   - fuel amount  

2. Run a loop for multiple time steps.  
3. At each step:  
   - calculate acceleration  
   - update velocity  
   - update altitude  
   - reduce fuel mass  

4. Store results in a Pandas DataFrame.  

This simulation generates a rocket altitude trajectory over time.
